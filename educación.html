Educación




Creemos que este producto podría ser llevado a las aulas y que podríamos ahondar en esta “caja negra” de la programación para poder crear nuevos contenidos en conjunto a nuestro estudiantado: agregar a nuestra currícula educativa no solo el aprender interfaces convenientes, sino el indagar, el generar nuestras propias interfaces, el explorar nuevas maneras de expresar y crear en el ámbitos de los nuevos medios. La programación web es tan solo uno de los tantos campos por explorar. A continuación dejamos algunos de los sitios que nos han ayudado en la construcción del nuestro. Información que creemos indispensable a la hora de transitar este camino con nuestro estudiantado:




  • w3schools sitio indispensable para comprender estructuras dentro del desarrollo web
  • MDN web docs sitio indispensable para comprender estructuras dentro del desarrollo web
  • Guía de Flexbox Esta guía es especialmente útil para comprender el posicionamiento de los elementos dentro de la página
  • Google fonts Agregado y busqueda de distintas fuentes/tipografías libres para nuestro sitio
  • Visual Studio Code Editor de código de preferencia. Intuitivo y simple.
  • Animista.net Este sitio nos sirve para crear animaciones en CSS
